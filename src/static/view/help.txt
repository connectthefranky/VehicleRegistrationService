Vehicle Registration Service Help

Configuration & Run
1) Ensure Java 17+ is installed.
2) Compile: javac src/Main.java src/server/*.java src/controller/*.java src/service/iface/*.java src/service/impl/*.java src/repository/Database.java src/repository/iface/*.java src/repository/impl/*.java src/model/*.java src/util/*.java
3) Run: java -cp src Main
4) Service will be available at http://localhost:8080

Endpoints
POST /account
  Body: {"accountId":"user@example.com"}
POST /register
  Header: Authorization: Basic base64(accountId:password)
  Body: {"registrationCode":"555K487","validUntil":"2024-11-18"}
GET /statistics
  Header: Authorization: Basic base64(accountId:password)
GET /registration/{registrationCode}
GET /gui (simple web UI)
